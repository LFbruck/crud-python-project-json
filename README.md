# CRUD (Python + JSON)

Desenvolvi um projeto simples de CRUD, em python usando JSON como um BD local

Algumas funcionalidades:
- Listar registros
- Criar registro
- Buscar registro por ID
- Atualizar registro
- Excluir registro

Tecnologias e Ferramentas usadas:

- Python 3.11+
- JSON
- Terminal / CLI
