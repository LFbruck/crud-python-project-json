 CRUD Python + JSON

Cirei um rojeto simples de CRUD - desenvolvido em Python, utilizando um arquivo JSON como armazenamento local.  



 Funcionalidades:

✔ Listar registros  
✔ Criar registro  
✔ Buscar registro por ID  
✔ Atualizar registro existente  
✔ Excluir registro  
✔ Armazenamento em `db.json`  
✔ Código simples e fácil de entender  



Tecnologias utilizadas:

- Python 3.11 +
- JSON
- PyCharm (JetBrains)
